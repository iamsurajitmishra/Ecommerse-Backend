# Backend Server
Perpose of this backend server is to communicate with the [frontend-angular-server](https://github.com/iamsurajitmishra/Ecommerse-Frontend)
and serve the request.

##Tech stack
 * Spring-boot
 * Spring-boot-JPA
 * Rest Repository
 * LOMBOK
 * Spring-web-mvc
 * Spring-rest
 * Spring-security
 * MySQL
